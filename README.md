# journal-project-v2-steps-api
Companion repository for a light-weight app to record a daily gratitude entry alongside other daily data. Repository is an API end point for recording daily steps.

For more information about how to use this repository alongside the full Journal App, see the [full repository](https://github.com/cheylea/journal-project-v2/blob/main/README.md).
